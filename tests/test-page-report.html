Test D: Attributes in CSV

  1. Run complete analysis
  2. Download Excel
  3. Verify each row has:
    • FLOW NAME: TEST_FLOW
    • PAGE NAME: page1
    • ERROR CODE: FONT-SIZE, ICON-SIZE, etc.
    • ELEMENT TYPE: button, span, div, etc.
    • ID: element IDs
    • TRACK ID: element trackids
    • DATA-TEST-ID: element data-testids
    • ERROR MESSAGE: descriptive message
